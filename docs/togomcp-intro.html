TogoMCP

Model Context Protocol Server for Life Sciences Data Integration

TogoMCP is a comprehensive MCP server that provides LLM agents with seamless access to a vast ecosystem of life sciences databases through SPARQL queries, RDF data exploration, and ID conversion services. It integrates over 20 major biological and biomedical databases, offering researchers a powerful toolkit for cross-database queries, data integration, and knowledge discovery.

Endpoint: https://togomcp.rdfportal.org/mcp

Developed by: DBCLS

What is TogoMCP?

TogoMCP bridges the gap between AI assistants and life sciences databases by implementing the Model Context Protocol (MCP). It enables researchers to query and analyze data from multiple biological databases without needing to understand complex query languages or database structures.

Key Features

  • 20+ Integrated Databases: Access UniProt, PubMed, ChEMBL, PDB, Ensembl, and many more
  • SPARQL Query Support: Execute powerful semantic queries across RDF databases
  • ID Conversion Services: Convert identifiers between different biological databases
  • Natural Language Interface: Query databases using conversational language through AI assistants
  • Cross-Database Integration: Combine data from multiple sources in a single query
  • Biomedical Entity Search: Find proteins, genes, diseases, and chemical compounds

Who Should Use TogoMCP?

  • Biologists and Medical Researchers: Quickly access and integrate data from multiple databases
  • Bioinformaticians: Build complex queries and data pipelines efficiently
  • Data Scientists: Perform integrative analyses across heterogeneous biological datasets
  • Clinical Researchers: Explore disease mechanisms and drug targets
  • Students: Learn about biological databases and data integration

Usage Examples

Here are real examples demonstrating TogoMCP's capabilities:

Example 1: Cardiovascular Disease Drug Target Discovery

Prompt: Find proteins that are associated with both cardiovascular diseases and have known small molecule inhibitors in ChEMBL, and classify them according to disease and drug availability.

Process: The AI agent used TogoMCP to:

  1. Search MeSH for cardiovascular disease terms
  2. Query ChEMBL for cardiovascular drug targets and inhibitors
  3. Execute SPARQL queries across ChEMBL RDF to find protein-inhibitor relationships
  4. Classify targets by disease category and development status
  5. Generate comprehensive reports and visualizations

Tools Used:

  • list_databases - Identify available databases
  • search_mesh_entity - Find disease classifications
  • search_chembl_target - Locate drug targets
  • get_MIE_file - Access ChEMBL schema documentation
  • run_sparql - Execute complex queries on ChEMBL RDF

Key Results:

  • Identified 75+ human proteins with validated small molecule inhibitors
  • Classified by disease: hypertension (15+ targets), coronary artery disease (20+ targets), heart failure (12+ targets)
  • Most productive target families: sodium channels (57 inhibitors), L-type calcium channels (27 inhibitors), beta-adrenergic receptors (25+ inhibitors)
  • Drug development status: 50+ approved drugs, 15+ in Phase 3 trials

Example 2: Human Enzyme Classification Analysis

Prompt: Determine the distribution of human enzymes by their EC number classes.

Process: The AI agent:

  1. Accessed UniProt RDF schema to understand enzyme classification
  2. Queried Swiss-Prot for all human enzymes with EC numbers
  3. Aggregated and counted enzymes by EC class
  4. Calculated distribution percentages
  5. Generated visualizations showing enzyme class distribution

Tools Used:

  • get_MIE_file - Access UniProt RDF schema
  • run_sparql - Query UniProt for enzyme data

Key Results:

  • Analyzed 4,442 unique human enzymes
  • Distribution: EC 2 Transferases (41.0%, 1,823 proteins), EC 3 Hydrolases (37.3%, 1,655 proteins), EC 1 Oxidoreductases (12.3%, 545 proteins)
  • Transferases and hydrolases dominate (78.3% combined), reflecting importance in cellular signaling and regulation
  • Smaller classes (EC 4-7) represent specialized but critical metabolic functions

Example 3: Natural Products vs Synthetic Compounds Analysis

Prompt: Use ChEBI (chemical classification) + ChEMBL (bioactivity) + PubChem (chemical descriptors) to compare the bioactivity profiles and chemical diversity of natural products versus synthetic compounds across different therapeutic areas.

Process: The AI agent:

  1. Queried ChEBI to identify natural product classifications
  2. Searched ChEMBL for bioactivity data of both natural and synthetic compounds
  3. Retrieved molecular descriptors from PubChem RDF
  4. Performed comparative statistical analysis
  5. Generated comprehensive reports on chemical and bioactivity differences

Tools Used:

  • get_MIE_file - Access schemas for ChEBI, ChEMBL, PubChem
  • run_sparql - Query across multiple RDF databases
  • get_compound_attributes_from_pubchem - Retrieve molecular descriptors

Key Results:

  • Natural Products: 70% show IC50 < 100 nM (ultra-high potency), average 8 chiral centers, hit 8.5 targets (polypharmacology), ~60% of anticancer drugs, higher sp3 fraction (0.6) for 3D diversity
  • Synthetic Compounds: 40% show IC50 < 100 nM, average 1 chiral center, hit 2-3 targets (high selectivity), dominant in cardiovascular/CNS/metabolic diseases, lower sp3 fraction (0.3)
  • Insight: Natural products excel in oncology with unique chemical space; synthetic drugs excel in target selectivity. Optimal strategy combines both approaches.

Setup Guide

TogoMCP can be integrated with various AI assistants. Below are setup instructions for popular platforms:

Claude Desktop

Claude Desktop supports TogoMCP through custom connectors (recommended for remote servers):

Method 1: Using Custom Connectors (Recommended)

  1. Open Claude Desktop and go to Settings → Connectors
  2. Click "Add custom connector" at the bottom of the section
  3. Enter the TogoMCP server URL: https://togomcp.rdfportal.org/mcp
  4. Click "Add" to complete the setup
  5. Enable the connector via the "Search and tools" button in your chat interface

Note: Custom connectors are available on Claude Desktop for users on Pro, Max, Team, and Enterprise plans.

Method 2: Manual Configuration (Alternative)

For advanced users, you can configure TogoMCP in the Claude Desktop configuration file:

  1. Locate the configuration file:
    • macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
    • Windows: %APPDATA%\Claude\claude_desktop_config.json
  2. Add TogoMCP configuration using the mcp-remote wrapper:
{ "mcpServers": { "togomcp": { "command": "npx", "args": [ "mcp-remote@latest", "--http", "https://togomcp.rdfportal.org/mcp" ] } } }

3. Restart Claude Desktop

4. The TogoMCP tools should now be available in your "Search and tools" menu

ChatGPT

ChatGPT supports MCP servers through Developer Mode (available for Business and Enterprise/Edu customers):

For Admins/Owners:

  1. Go to Workspace Settings → Permissions & Roles → Connected Data
  2. Enable "Developer mode / Create custom MCP connectors"

Adding TogoMCP Connector:

  1. In the chat panel, click the "+" icon
  2. Select "Developer Mode" from the dropdown
  3. Click "Add sources"
  4. Enter the MCP server details:
    • Name: TogoMCP
    • URL: https://togomcp.rdfportal.org/mcp
  5. Click "Create" and authorize the connector

Using TogoMCP in Conversations:

  • Select TogoMCP from the tools menu in Developer Mode
  • Be explicit in your prompts: "Use TogoMCP to find proteins related to diabetes"
  • ChatGPT will show tool call confirmations before executing write actions

Note: Developer Mode is currently in beta and requires Pro, Team, or Enterprise plans.

Gemini CLI

Gemini CLI supports MCP servers through Streamable HTTP transport. TogoMCP uses Streamable HTTP, not SSE.

Adding TogoMCP:

  1. Use the Gemini CLI command to add the server:
gemini mcp add --transport http togomcp https://togomcp.rdfportal.org/mcp

Manual Configuration:

Alternatively, edit your Gemini CLI settings file directly:

  • User config: ~/.gemini/settings.json
  • Project config: .gemini/settings.json

Add the following configuration:

{ "mcpServers": { "togomcp": { "httpUrl": "https://togomcp.rdfportal.org/mcp" } } }

Verifying the Setup:

  1. Start Gemini CLI
  2. Use the /mcp command to list available servers
  3. TogoMCP should appear in the list with its available tools

Note: Gemini CLI automatically discovers tools from the MCP server and makes them available for use in conversations.

Available Databases

TogoMCP provides access to the following life sciences databases through RDF and SPARQL:

UniProt

Comprehensive protein sequence and functional information with 444M proteins. Swiss-Prot (manually curated) and TrEMBL (automatically annotated). Essential for protein research with cross-references to 200+ databases.

PubMed

Bibliographic information for biomedical literature from MEDLINE. Includes publication metadata, abstracts, authors, MeSH term annotations, and cross-references. Essential for literature searches.

PubTator

Biomedical entity annotations from PubMed literature using text mining and manual curation. Contains Disease and Gene annotations linked to PubMed articles for literature-based discovery.

ChEMBL

Manually curated database of bioactive molecules with 2.4M+ compounds, 1.6M assays, and 20M bioactivity measurements. Essential for drug discovery and medicinal chemistry.

PubChem

Comprehensive public database with 119M compounds, 339M substances, and 1.7M bioassays. Includes molecular descriptors, bioactivity data, and extensive cross-references.

ChEBI

Ontology database of 217,000+ chemical entities of biological interest with hierarchical classification, molecular data, biological roles, and chemical relationships.

PDB (Protein Data Bank)

3D structural data for proteins, nucleic acids, and complexes with 204K+ entries from X-ray, NMR, and cryo-EM. Essential for structural biology and drug design.

Ensembl

Comprehensive genomics database with genome annotations for 100+ species. Contains genes, transcripts, proteins, and exons with genomic locations and quality annotations.

NCBI Gene

Gene database with 57M+ entries covering protein-coding genes, ncRNAs, pseudogenes across all organisms. Includes chromosomal locations and orthology relationships.

Reactome

Open-source curated knowledgebase of biological pathways with 22,000+ pathways across 30+ species. Includes molecular interactions, biochemical reactions, and disease associations.

Rhea

Expert-curated database of 17,078 biochemical reactions. All reactions are atom-balanced, chemically annotated, and linked to standard compound vocabularies (ChEBI).

Gene Ontology (GO)

Controlled vocabulary with 30,804 biological process terms, 12,793 molecular function terms, and 4,568 cellular component terms for standardized gene annotation.

MeSH (Medical Subject Headings)

National Library of Medicine's controlled vocabulary thesaurus for biomedical literature indexing with hierarchical structure across 16 main categories.

MONDO (Monarch Disease Ontology)

Comprehensive disease ontology integrating multiple databases with cross-references to OMIM, Orphanet, DOID, MeSH, ICD, and 35+ databases.

NANDO

Comprehensive ontology for Japanese intractable diseases with 2,777 disease classes, multilingual labels, and cross-references to international disease ontologies.

ClinVar

Aggregates genomic variation and its relationship to human health with 3.5M+ variant records, clinical interpretations, and disease associations.

MedGen

NCBI's portal for medical genetics with 233,000+ clinical concepts integrating OMIM, Orphanet, HPO, and MONDO for variant interpretation.

NCBI Taxonomy

Comprehensive biological taxonomic classification covering 3M+ organisms with hierarchical relationships, scientific/common names, and genetic code assignments.

BacDive

Bacterial Diversity Metadatabase with 97,000+ strain records covering taxonomy, morphology, physiology, cultivation conditions, and molecular data.

MediaDive

Microbial culture media database with 3,289 standardized recipes from DSMZ for bacteria, archaea, fungi, yeast, microalgae, and phages.

DDBJ

DNA Data Bank of Japan providing nucleotide sequence data from INSDC with genomic annotations and taxonomic classification.

GlyCosmos

Comprehensive glycoscience portal integrating glycan structures, glycoproteins, glycosylation sites, glycogenes, and lectin-glycan interactions.

Available Tools

TogoMCP provides the following categories of tools for database interaction:

Database Discovery & Schema

list_databases

List all available databases with descriptions. Essential first step to understand what data sources are accessible.

get_MIE_file

Retrieve the MIE (Model, Instance, Example) file containing ShEx schema, RDF structure, and SPARQL examples for a specific database. Critical for understanding how to query each database.

get_graph_list

Get a list of named graphs in a specific RDF database for more granular data access.

get_sparql_endpoints

Retrieve the available SPARQL endpoints for RDF Portal databases.

get_sparql_example

Get example SPARQL queries for a specific database to learn query patterns.

SPARQL Query Execution

run_sparql

Execute a SPARQL query on a specific RDF database. The primary tool for retrieving and analyzing data from biological databases.

Entity Search & Lookup

search_uniprot_entity

Search for UniProt protein entity IDs by keywords (e.g., protein names, gene names).

search_chembl_target

Search for ChEMBL drug targets by keywords.

search_chembl_molecule

Search for ChEMBL molecules (compounds) by keywords.

search_chembl_id_lookup

Search for ChEMBL IDs by query string for general entity lookup.

search_mesh_entity

Search for MeSH (Medical Subject Headings) terms for diseases, chemicals, and biomedical concepts.

search_pdb_entity

Search for PDB entries, chemical components, or biologically interesting molecules by keywords.

Chemical Data Retrieval

get_pubchem_compound_id

Get PubChem Compound ID from a compound name (e.g., "resveratrol").

get_compound_attributes_from_pubchem

Retrieve detailed compound attributes from PubChem RDF including molecular descriptors, properties, and classifications.

ID Conversion (TogoID Integration)

togoid_getAllDataset

Get configuration for all available datasets in TogoID, including labels, regex patterns, prefixes, and examples.

togoid_getDataset

Get configuration for a specific dataset including validation patterns and available annotations.

togoid_convertId

Convert IDs from one database to another (e.g., UniProt to PDB, Ensembl to NCBI Gene). Supports multi-step conversions.

togoid_countId

Count how many IDs can be converted between two databases.

togoid_getAllRelation

Get all possible conversion relationships between databases.

togoid_getRelation

Get relationship details between two specific databases.

togoid_getDescription

Get descriptions for all databases in TogoID with multilingual support.

General Guidance

RDF_Portal_Guide

Get general guidelines for using the RDF Portal and understanding its structure.

Related Resources

TogoMCP Server

The main endpoint for TogoMCP. Access comprehensive MCP server documentation, available tools, and integration examples. This server implements the Model Context Protocol standard for seamless integration with AI assistants.

RDF Portal

NBDC RDF Portal is a comprehensive repository for semantic data in life sciences, developed by DBCLS and the National Bioscience Database Center (NBDC). It hosts 20+ RDF datasets comprising over 45 billion triples, providing SPARQL endpoints, metadata, and downloadable RDF files. The portal ensures dataset quality through reviews and conformance to RDFizing guidelines, promoting interoperability and FAIR principles in biological data.

TogoID

TogoID is an identifier conversion service that bridges biological datasets across diverse categories. It supports multi-step conversions between 65+ databases including genes, proteins, compounds, diseases, and more. Features include semantic representation of relationships, a user-friendly web interface, API access, and automatic weekly updates. TogoID is essential for integrative analysis and connecting data from heterogeneous sources.

PubDictionaries MCP Server

PubDictionaries MCP server provides access to biomedical term dictionaries and annotation services. It works seamlessly with TogoMCP for comprehensive biomedical text mining and entity recognition. Combine both servers to perform literature mining alongside database queries for powerful knowledge discovery workflows.

DBCLS (Database Center for Life Science)

DBCLS is a research institute within ROIS (Research Organization of Information and Systems) focused on developing technologies for database integration in life sciences. DBCLS organizes the annual BioHackathon series, maintains numerous biological databases and tools, and promotes Semantic Web technologies in bioinformatics. Key projects include TogoID, TogoMCP, RDF Portal, and various data integration platforms that enable researchers to seamlessly access and analyze heterogeneous biological data.
